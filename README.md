# React + Vite

# 项目名称

TokenB后台管理系统

# 项目简介

TokenB后台管理系统，主要包含系统/商户/邀请/闪兑/币币/财务/应用端管理/数据管理模块

## 技术栈

- **前端框架**: React
- **构建工具**: Vite
- **CSS**: LESS
- **其他**: React Router，ant-design, prettier,Eslint
